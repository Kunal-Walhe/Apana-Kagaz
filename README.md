# Apana Kagaz

<div align="center">

  <p>A serene, elegant web application dedicated to poetry, shayaris, and ghazals. Built with React and Vite, Apana Kagaz offers a quiet place for lovers of words to explore, save, and write beautiful verses.</p>

## ✨ Features

- **Explore Poetry:** Browse through a rich collection of poems, ghazals, and shayaris.
- **Advanced Search & Filtering:** Quickly find poetry by keywords, or filter by specific authors and categories.
- **Favorites System:** Save the poetry that touches your heart and view them anytime in your favorites list.
- **Write Your Own (Likhein):** Contribute to your local collection by adding your own verses.
- **Elegant & Responsive UI:** A carefully crafted, highly aesthetic, and fully responsive user interface featuring modern touches like backdrop blur, custom animations, and a soothing color palette.

## 🛠️ Technologies Used

- **Framework:** React 19 (TypeScript)
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Build Tool:** Vite

## 🚀 Getting Started

Follow these instructions to run the project on your local machine.

### Prerequisites
- Node.js (version 18 or higher recommended)

### Installation & Setup

1. **Clone the repository** (if you haven't already) and navigate to the project directory:
   ```bash
   cd "Apana Kagaz"
   ```

2. **Install the dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Open your browser** and visit the URL provided in the terminal (usually `http://localhost:5173` or `http://localhost:3000`) to view the application.

## 📁 Project Overview

- `App.tsx`: The main application component containing the primary UI, routing logic, and state management.
- `components/`: Contains reusable React components like `PoetryCard` and `PoetryForm`.
- `data.json` / `data.ts`: The initial collection of poetry data loaded into the application.
- `types.ts`: TypeScript interfaces and type definitions used across the project.
